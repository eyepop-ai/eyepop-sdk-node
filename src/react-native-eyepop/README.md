# react-native-eyepop

The official Typescript library for EyePop.ai's inference API for React Native

## Installation

```sh
npm install react-native-eyepop
```

## Usage


```js
import { multiply } from 'react-native-eyepop';

// ...

const result = await multiply(3, 7);
```
